# vRooms
vRooms is a room based video chat application made with Vue.js, Vuetify and the Jitsi IFrame API.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
